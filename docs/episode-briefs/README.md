# Episode Briefs

Planning briefs for upcoming episodes, per the
[episode-brief skill](../../.github/skills/episode-brief.md). Each file is a
dispatch-ready prompt for the sub-agent fleet, held here in DRAFT status until the
showrunner greenlights the episode. Aired episodes graduate to
[docs/exec-reports/](../exec-reports/) once the work ships and the exec report is
written.
