## Install

The v2 line ships as `az-ai-v2` during the dual-tree window. v1 remains
available as `az-ai` (see [v1 (legacy)](#v1-legacy) below).

### Homebrew (macOS, Linux)

```bash
brew tap schwartzkamel/tap
brew install az-ai-v2
```

### Scoop (Windows)

```powershell
scoop bucket add schwartzkamel https://github.com/SchwartzKamel/scoop-bucket
scoop install az-ai-v2
```

### Nix (flake)

```bash
nix run github:SchwartzKamel/azure-openai-cli#az-ai-v2
```

Or, for a persistent install:

```bash
nix profile install github:SchwartzKamel/azure-openai-cli#az-ai-v2
```

### Docker (GHCR)

Secondary option — native AOT is recommended for latency-sensitive use.

```bash
docker pull ghcr.io/schwartzkamel/azure-openai-cli/az-ai-v2:2.0.0
docker run --rm --env-file .env \
  ghcr.io/schwartzkamel/azure-openai-cli/az-ai-v2:2.0.0 "Hello world"
```

### Pre-built binaries

Download from [Releases](https://github.com/SchwartzKamel/azure-openai-cli/releases/tag/v2.0.0):

| Platform                   | Artifact                                           |
|----------------------------|----------------------------------------------------|
| Linux x64 (glibc)          | `azure-openai-cli-v2-linux-x64.tar.gz`             |
| Linux x64 (musl / Alpine)  | `azure-openai-cli-v2-linux-musl-x64.tar.gz`        |
| macOS x64                  | `azure-openai-cli-v2-osx-x64.tar.gz`               |
| macOS arm64                | `azure-openai-cli-v2-osx-arm64.tar.gz`             |
| Windows x64                | `azure-openai-cli-v2-win-x64.zip`                  |

> **arm64 note.** Linux arm64 and Windows arm64 matrices roll forward
> to 2.0.1. If arm64 is your primary platform, pin `v1.9.1` until then.

### v1 (legacy)

v1 is still supported on the 1.9.x line for users not yet on MAF.
Install `az-ai` from the pinned v1 manifests:

```bash
# Homebrew (pinned v1)
brew install schwartzkamel/tap/azure-openai-cli@1.9.1

# Scoop (pinned v1)
scoop install schwartzkamel/azure-openai-cli@1.9.1

# Direct download
# https://github.com/SchwartzKamel/azure-openai-cli/releases/tag/v1.9.1
```

Both binaries install side by side; they read the same env vars, the
same config file, and the same `.squad/` memory. See
[`docs/migration-v1-to-v2.md`](docs/migration-v1-to-v2.md) for the
upgrade story.
```
